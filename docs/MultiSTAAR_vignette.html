A statistical framework for powerful multi-trait rare variant analysis in large-scale whole-genome sequencing studies

Xihao Li and Zilin Li

2024-06-15

The MultiSTAAR package accompanies the paper “A statistical framework for powerful multi-trait rare variant analysis in large-scale whole-genome sequencing studies” and is designed for detecting rare variant associations with multiple phenotypes in whole-genome sequencing studies.

Note: The following results are depending on R 4.2.3. The results could be different on other R versions due to different random number generator.

Data set-up: Generating signal regions, covariates, family-structure

The genotype data was generated by simulating 10,000 samples for a 1 megabase (Mb) region using the calibration coalescent model (COSI). The data were generated to mimic the linkage disequilibrium structure of an African-American population by using COSI. We analyzed 30,095 low frequency and rare variants (minor allele frequency, MAF < 0.05) in the sequence.

We simulated 2,500 families and each family has 4 individuals. The kinship matrix is generated as follows:

## kinship
grid <- 1
Npercell <- 10000
ndiv <- 1
vfam <- 0.5
N <- round(grid*grid*Npercell/ndiv)

unitmat <- matrix(0.5, 4, 4)
diag(unitmat) <- 1
unitmat[1,2] <- unitmat[2,1] <- 0
ped <- data.frame(famid = rep(as.integer(1:2500), each=4), id = as.integer(1:10000L), fa = rep(0, 10000), mo = rep(0, 10000))
for(i in 1:2500) {
  ped$fa[4*i-(0:1)] <- ped$id[4*i-3]
    ped$mo[4*i-(0:1)] <- ped$id[4*i-2]
}
kins <- makekinship(ped$famid, ped$id, ped$fa, ped$mo)

We generated two covariates, one is continuous, the other one is binary.

set.seed(800)

We calculated the Phred score of the 10 annotations. Note that the input of annotations for MultiSTAAR function was at Phred scale (annotation_phred). In practice, the Phred score is calculated based on all variants across the genome, available from the FAVOR database (https://favor.genohub.org).

rank <- colRanks(Z,preserveShape = TRUE)
PHRED <- -10*log10(1-rank/dim(rank)[1])

We randomly selected a 5-kilobase (kb) signal region (genotype-phenotype association regions) across the 1-Mb region.

# Length of the sequence
maxLength <- 1000000
# Length of the signal region
sigLength <- 5000
# Location of the signal region
startloc <- sample(1:(maxLength-sigLength+1),1)
endloc <- startloc + sigLength - 1
# Extract the signal region from snploc
snplist <- which(snploc$CHROM_POS >= startloc & snploc$CHROM_POS <= endloc)
snpRegion <- snploc[snplist,]
# number of variants in the signal region
numSNPs <- dim(snpRegion)[1]
# genotype of the signal region
Geno <- genotype[,snplist]

We first randomly select 5 annotations to determine the probability of variants being causal in the region through a logistic model. We then randomly selected casual variants based on these probabilities. More details could be found in the Methods Section.

# Simulate causal variants from this region
b0 <- rje::logit(0.015)
b <- rep(log(5),10)

causalprob <- apply(Z[snplist,],1,function(z){
  ind <- sample(1:10,5)
  rje::expit(b0 + b[ind] %*% z[ind])
})
isCausal <- rbinom(numSNPs,1,causalprob)

Analyzing the simulated data using MultiSTAAR

Unrelated samples

To illustrate the MultiSTAAR method for multi-trait anlaysis, we consider the joint analysis of 3 phenotypes. The phenotype of unrelated samples was generated through a multivariate linear model (MLM), with the phenotypic correlation structure mimicking that of the lipid trait analysis of LDL-C, HDL-C, and triglycerides from real data, that is, \[ \boldsymbol{\Sigma} = \text{Cov}\left(\boldsymbol{Y}\right) = \text{Cov}\left(\begin{array}{c} Y_1 \\ Y_2 \\ Y_3 \end{array}\right) = \left[\begin{array}{ccc} 1 & -0.1 & 0.2 \\ -0.1 & 1 & -0.4 \\ 0.2 & -0.4 & 1 \end{array}\right] \] Without loss of generality, we set the causal variants to affect two positively correlated phenotypes (i.e. \(Y_1\) and \(Y_3\)), and set the effect sizes as a decreasing function of MAFs, beta=-0.1*log10(MAF). The coefficients for the causal variants were all positive.

# effects size of covariates 
alpha0 <- 0
alpha1 <- 0.5
alpha2 <- 0.5
# effect size of causal variants
c0 <- 0.1
mu <- rep(0, 3)
Sigma <- matrix(c(1, -0.1, 0.2, -0.1, 1, -0.4, 0.2, -0.4, 1), nrow = 3)
# Generate error distributions
eps <- MASS::mvrnorm(N, mu = mu, Sigma = Sigma)
beta <- -c0 * log10(maf[snplist][which(isCausal==1)])
# generate multi-phenotype data
Y1 <- alpha0 + alpha1 * X1 + alpha2 * X2 + as.matrix(genotype)[,snplist][,which(isCausal==1)] %*% beta + eps[,1]
Y2 <- alpha0 + alpha1 * X1 + alpha2 * X2 + eps[,2]
Y3 <- alpha0 + alpha1 * X1 + alpha2 * X2 + as.matrix(genotype)[,snplist][,which(isCausal==1)] %*% beta + eps[,3]

We applied MultiSTAAR to analyze the simulated data set. We first fit the null model using MLM.

# Fitting MultiSTAAR null model for unrelated samples using fit_null_glmmkin_multi()
pheno <- data.frame(Y1=Y1,Y2=Y2,Y3=Y3,X1=X1,X2=X2,id=c(1:N))
obj_nullmodel <- MultiSTAAR::fit_null_glmmkin_multi(cbind(Y1,Y2,Y3)~X1+X2,data=pheno,family=gaussian(link="identity"),kins=NULL,id="id")
## [1] "kins is NULL, fit multivariate linear model."
## Fitting a multiple phenotype analysis model...

We then applied MultiSTAAR to test for the association.

pvalues <- MultiSTAAR(genotype=Geno,obj_nullmodel=obj_nullmodel,annotation_phred=PHRED[snplist,],rare_maf_cutoff=0.05)

Related samples

The phenotype of related samples was generated through a multivariate linear mixed effects model (MLMM).

set.seed(800)
## random effect
randomfam <- mvrnorm(N/4*3, rep(0, 4), vfam * unitmat)
randomfam <- matrix(as.vector(t(randomfam)),ncol=3)

# generate multi-phenotype data
Y1 <- alpha0 + alpha1 * X1 + alpha2 * X2 + as.matrix(genotype)[,snplist][,which(isCausal==1)] %*% beta + randomfam[,1] + eps[,1]
Y2 <- alpha0 + alpha1 * X1 + alpha2 * X2 + randomfam[,2] + eps[,2]
Y3 <- alpha0 + alpha1 * X1 + alpha2 * X2 + as.matrix(genotype)[,snplist][,which(isCausal==1)] %*% beta + randomfam[,3] + eps[,3]
